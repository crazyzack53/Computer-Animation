Description

This project creates a dynamic physics-based domino animation using a Python script inside Maya. The script:

  • Prompts for an image file (20×20).
  • Creates over 200 dominoes based on the pixels.
  • Colors each domino according to the image.
  • Sets up rigid body dynamics and a side pusher.
  • Generates a full 360-frame simulation automatically.
